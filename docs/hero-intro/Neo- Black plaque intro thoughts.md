When This Works (and When It Fails)
-----------------------------------

### ✅ It works if:

*   It lasts **≤ 1.5–2.0 seconds** total
*   It is **non-interruptive** (no “loading…” text)
*   It feels like an _invocation_, not a brand flex
*   It never replays once dismissed (cookie/localStorage)

### ❌ It fails if:

*   It blocks scroll
*   It repeats every visit
*   It competes with performance metrics
*   It feels like a movie trailer

This is not a splash screen.

If someone notices it, it should feel deliberate and calm.

Recommended Execution (Very Specific)
-------------------------------------

### Phase 1: Invocation (0.8–1.2s)

*   Parchment background only
*   Black marble plaque
*   Text:

No logo.

No motion except a _very subtle_ ambient grain or light falloff.

### Phase 2: Dissolve (0.6–0.8s)

*   Plaque **fades**, not slides
*   Parchment subtly darkens
*   Hero content fades _in underneath_, already laid out

This matters:  
The system appears _as if it was always there_.

### Phase 3: Lockout

*   Store `hasSeenInvocation = true`
*   Never show it again unless:
    *   hard refresh + modifier key
