# Blueprint Images

Responsive assets for the Blueprint page. Naming follows project convention: `-small` (400w), `-medium` (800w), `-large` (1200w).

| File | Use |
|------|-----|
| `blueprint-background-*.webp` | Background layer for Blueprint map |
| `blueprint-map-nomothetesis-*.webp` | Nomothetēsis process map (with hotspot regions) |
| `blueprint-pnyx-interior-*.webp` | Pnyx/Ekklesia interior imagery |
| `blueprint-ekklesia-hub-owl.webp` | Ekklesia hub with owl motif |
| `blueprint-ekklesia-amphitheater.webp` | Amphitheater reference |
| `*.png` | Source files for future editing |

**Note:** `blueprint-pnyx-interior-800-dup.webp` was from the `Pnyx_inerior` typo—different file size than medium; delete if redundant.
